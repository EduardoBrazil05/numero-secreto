# Jogo do Número Secreto

## Sobre o projeto

Este projeto consiste em um jogo interativo desenvolvido com JavaScript, onde o usuário deve tentar descobrir um número aleatório gerado pelo sistema.

A cada tentativa, o site informa se o número secreto é maior ou menor que o valor enviado pelo usuário, criando uma experiência dinâmica de lógica e adivinhação.

Quando o jogador acerta o número, o sistema exibe a quantidade de tentativas realizadas e libera a opção de reiniciar a partida.

Neste projeto fui responsável pelo desenvolvimento de toda a lógica da aplicação utilizando JavaScript.

A interface frontend foi disponibilizada previamente como base do desafio, permitindo foco total na construção das regras de negócio e interações da aplicação.

---

## Funcionalidades

- Geração aleatória de números
- Verificação de tentativas do usuário
- Dicas indicando se o número secreto é maior ou menor
- Contador de tentativas
- Reinício do jogo
- Bloqueio/desbloqueio do botão de reiniciar
- Conversão de texto em voz utilizando Web Speech API
